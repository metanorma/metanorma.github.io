To install, run:

[source,sh]
----
brew tap metanorma/metanorma
brew install metanorma
----

Alternatively:

[source,sh]
----
brew install metanorma/metanorma/metanorma
----

Or

[source,sh]
----
brew install --HEAD https://raw.githubusercontent.com/metanorma/homebrew-metanorma/master/metanorma.rb
----
