== Metanorma for IHO output formats

=== General

Metanorma for IHO supports the generation of the following formats.

`html`:: HTML output
`pdf`:: PDF output
`doc`:: Word document in the `.doc` extension
`xml`:: Metanorma semantic and presentational XML
